Chat application

Fullstack application development

I developed a chat application using React.js for the UI design, socket.io for the server side. the dependencies includes express, nodemon, react-dom, react-emoji, react-scroll-to-bottom, socket.io-client, socket.io, and cores. I added some custome CSS styling to make make it look good.

features

The user joins the chatroom by inputting the their name and the name of the room the admin welcomes the new user with a greeting stating that there is a new user in the room. the users can exchange messages with each other.
